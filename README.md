LiqueBot
====

Este repositorio contiene todo el software de control del Proyecto LiqueBot. 

Tanto el programa de control embebido en el robot que contiene las librerías necesarias para el control de los
servomotores permitiendo el avance, retroceso y distintos modos de cambio de trayectoria, como el código necesario
para tomar, enviar y almacenar las lecturas.

Tambíén incluye el software de conexión a la estación base y la APP para controlarlo.
